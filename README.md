# ADS_Application

一个视频生成软件的前端。
软件功能的大致描述：使用该软件，用户可以自主上传图片、文字等基础素材，软件会将这些基础素材综合在一起，计算出一个符合用户预期的生成素材（可以是图片、物体模型等），并将该生成素材插入到某个视频中用户期望的地方，使之成为视频的一部分。
前端包括以下几个功能：视频选择、视频区间选择、插入区域选择、图片和文字等基本素材的上传、生成素材的展示和选择、生成视频的展示和保存、与后端通信（REST）。
